# tail-tip

Tailwind/Flowbite-friendly TipTap editor wrapper (vanilla JS) with toolbar bindings.

## Install

```bash
npm i tail-tip @tiptap/core @tiptap/starter-kit
npm i @tiptap/extension-bold @tiptap/extension-highlight @tiptap/extension-underline @tiptap/extension-link
npm i @tiptap/extension-text-align @tiptap/extension-image @tiptap/extension-youtube
npm i @tiptap/extension-text-style @tiptap/extension-font-family @tiptap/extension-color
```

## Local Development

```bash
npm install
npm run typecheck
npm run build
```
